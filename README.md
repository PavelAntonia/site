# Portfolio Site

A personal portfolio website for Antonia Pavel, showcasing experience, skills, and CV.

## Getting Started

```bash
pnpm install
pnpm dev
```

## Running with Docker

You can run the app in a Docker container:

```bash
docker build -t antonia-site . && docker run -p 3000:3000 antonia-site
```

Then open [http://localhost:3000](http://localhost:3000) in your browser.

## Features
- Project showcase
- Skills section
- Downloadable CV
- Contact form
- Responsive design
